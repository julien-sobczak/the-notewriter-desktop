The files `icon.svg` and `icon.png` are copied from exported resources in repository `the-notewriter`:

```shell
$ cp ~/Workshop/the-notewriter/website/design/export/favicon.svg  resources/icon.svg
$ cp ~/Workshop/the-notewriter/website/design/export/favicon@3x.png  resources/icon.png
```

⚠️ The PNG file must be at least `512x512`.
